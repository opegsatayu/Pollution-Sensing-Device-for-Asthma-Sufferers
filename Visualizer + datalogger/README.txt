This script should only be executed after:
a) The warning system and sensor setup code have been uploaded onto the microcontroller
b) Your laptop is successfully connected to the microcontroller.

Once run, the program will display:
1.) real-time visualization of the pollutant reading from the sensor (color coded)
2.) Datalog and average the sensor readings into the designated Excel file 

Once closed, the program will display: 
1.) Graph of exposure vs time of each sensor (color coded)
